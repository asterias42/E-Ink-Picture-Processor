# E-Ink-Picture-Processor
Crops, scales and dithers pictures to use them on an e-ink display.

The whole picture processing is executed in the browser. There is no upload of any data!

Works on desktop and mobile browsers.

To use this tool either download the html and open it with your browser, or just visit:

https://asterias42.github.io/E-Ink-Picture-Processor/
